Simple CLI Calculator

A beginner-friendly Python command-line calculator that performs basic arithmetic operations with proper error handling.

Features
Addition
Subtraction
Multiplication
Division
User input support
Invalid input handling
Division by zero protection
Technologies Used
Python 3
Project Structure
simple-calculator/
│
├── calculator.py
└── README.md
How It Works

The program:

Takes two numbers as input
Asks the user to choose an operator
Performs the selected calculation
Displays the result in the terminal

Supported operators:

+
-
*
/
Installation & Run
Clone Repository
git clone <your-repository-link>
Navigate to Project Folder
cd simple-calculator
Run the Program
python calculator.py
